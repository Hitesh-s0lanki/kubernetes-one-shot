# Kubernetes - One Shot

An open-source container orchestration platform. It automates the deployment, scaling, and management of containerized applications.

## Kubernetes Architecture

![Kube-Arch](/images/kube-arch.png)

## Type of Cluster For Kubernetes

- Kubeadm
- minikube
- kind (docker)
- EKS/AKS/GKS

## Workloads

Deployment, StatefulSets, DaemonSets, ReplicaSets, Jobs, CronJobs

## Networking

Cluster Networking, Services, Ingress, Network polices

## Storage

Persistent Volumes (PV), persistent Volume Claims (PVC), StorageClasses, ConfigMaps, Secrets

## Scaling and Scheduling

HPA (Horizontal Pod Scaling), VPA (Vertical Pod Scaling), Node Affinity, Taints/Tolerations, Resource Quotas, limits, Probes

## Cluster Administration

RBAC (Role Base Access Control), Cluster Upgrade, Custom Resource Definitions (CRDs).

## Monitoring and Logging

Metrics Server, Logging, Monitoring tools

## Advanced Features

Operatores, Helm, Service Mesh, kubernetes API

## Security

pod Security Standards, Image Scanning, Network polivies, Secrets Encryption

## Cloud-Native Kubernetes

Managed Services (EKS), Cluster Autoscaler, Spot/Preemptible Nodes

## Debugging and Troubleshooting

kubectl Debugging, Logs, Resource Usage Analysis.
